## DETAILED STYLE GUIDELINES FOR ALL LOGOS:

### Color Specifications:
```
Primary: Electric Cyan #00f0ff
Secondary: Electric Green #00ff88
Accent variations: #00d4ff (light cyan), #00ffaa (light green)
Background (if needed): #1a1a1a or transparent
Glow effects: Use outer glow with 10-20px radius at 40-60% opacity
```

### Font Recommendations:
- **Modern Sans-Serif:** Poppins, Inter, Montserrat, Space Grotesk
- **Weight:** 600-700 (SemiBold to Bold)
- **Avoid:** Thin weights, script fonts, overly decorative fonts

### Shopping Cart Icon Style:
- **Geometric:** Use circles, rectangles, and clean lines
- **Simplified:** 3-5 main shapes maximum
- **Recognizable:** Instantly identifiable as a shopping cart
- **Modern:** Avoid realistic/3D cart, prefer flat/isometric style
- **Tech-enhanced:** Could include subtle circuit lines or nodes

### Technical Requirements:
- **Format:** PNG with transparency
- **Resolution:** 2x size for retina displays
- **Edge quality:** Sharp, anti-aliased, vector-quality
- **Glow effects:** Subtle, not overpowering
- **Scalability:** Test at 16x16px to ensure clarity

## GEMINI GENERATION TIPS:

1. **For best results with Gemini:**
   - Start with "Create a professional logo design for a tech brand"
   - Include exact color codes (#00f0ff, #00ff88)
   - Specify "transparent background, PNG format"
   - Request "high resolution, vector-quality edges"
   - Add "modern, minimal, tech aesthetic"

2. **If first generation isn't perfect:**
   - Ask for "more minimal" or "more detailed"
   - Request "adjust colors to be brighter/darker"
   - Specify "make text more readable" if needed
   - Try "add more glow effect" or "reduce glow"

3. **For icon designs:**
   - Emphasize "simple geometric shapes"
   - Request "works at small sizes"
   - Add "bold, recognizable silhouette"

4. **For text/wordmark:**
   - Specify "modern sans-serif, similar to [font name]"
